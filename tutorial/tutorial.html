Tutorial#

There is a simple example that how to extend our pointrix framework to dynamic gaussaian, which the deformation is generated by MLP which take time step as input.

Add your model#

First, we need to import base model from pointrix so that we can inherit, registry and modify them.

from pointrix.model.base_model import BaseModel, MODEL_REGISTRY

Then, we need to implemet our model based BaseModel which contains full gaussian point implemetation.

Note

Your can refer to pointrix/model/base_model.py for more detail if you care about the full gaussian point implemetation.

We highlight the modified part.#
 1@MODEL_REGISTRY.register()
 2class DeformGaussian(BaseModel):
 3    def __init__(self, cfg, datapipeline, device="cuda"):
 4        super().__init__(cfg, datapipeline, device)
 5        self.deform = DeformNetwork(is_blender=False).to(self.device)
 6        self.time_interval = 1. / datapipeline.training_dataset_size
 7        self.smooth_term = get_linear_noise_func(
 8            lr_init=0.1, lr_final=1e-15, lr_delay_mult=0.01, max_steps=20000)
 9
10    def forward(self, batch, step=-1, training=False):
11        N = len(self.point_cloud.position)
12        camera_fid = torch.Tensor([batch[0]['camera'].fid]).float().to(self.device)
13        position = self.point_cloud.get_position
14        time_input = camera_fid.unsqueeze(0).expand(position.shape[0], -1)
15        ast_noise = torch.randn(1, 1, device='cuda').expand(N, -1) * self.time_interval * self.smooth_term(step)
16        if step < 3000 and training:
17            d_xyz, d_rotation, d_scaling = 0.0, 0.0, 0.0
18        else:
19            if training:
20                d_xyz, d_rotation, d_scaling = self.deform(position.detach(), time_input+ast_noise)
21            else:
22                d_xyz, d_rotation, d_scaling = self.deform(position.detach(), time_input)
23
24        render_dict = {
25            "position": self.point_cloud.get_position + d_xyz,
26            "opacity": self.point_cloud.get_opacity,
27            "scaling": self.point_cloud.get_scaling + d_scaling,
28            "rotation": self.point_cloud.get_rotation + d_rotation,
29            "shs": self.point_cloud.get_shs,
30        }
31        
32        return render_dict

Add your dataset#

To add our dataset, we need to inhert BaseReFormatData class and rewrite load_camera and load_pointcloud.

First, we need to import base data format from pointrix so that we can inherit, registry and modify them.

from pointrix.dataset.base_data import DATA_FORMAT_REGISTRY, BaseReFormatData, SimplePointCloud
from pointrix.camera.camera import Camera

Then, we need to implement the function to load the camera in dataset.

Note

Pointrix support common dataset reading so you do not need to implement by yourself in the most cases. There is just an example to illustrate how to add your own dataset.

We highlight the modified part.#

16        xyz = np.load(os.path.join(self.data_root, "points.npy"))

Modify Trainer#

In deformable gaussian, we need to pass the value of global step to model, so we can inherit it and add our own operation:

We highlight the modified part.#
 1from pointrix.engine.default_trainer import DefaultTrainer
 2class Trainer(DefaultTrainer):
 3
 4    def train_step(self, batch: List[dict]) -> None:
 5        """
 6        The training step for the model.
 7
 8        Parameters
 9        ----------
10        batch : dict
11            The batch data.
12        """
13        render_dict = self.model(batch, step=self.global_step, training=True)
14        render_results = self.renderer.render_batch(render_dict, batch)
15        self.loss_dict = self.model.get_loss_dict(render_results, batch)
16        self.loss_dict['loss'].backward()
17        self.optimizer_dict = self.model.get_optimizer_dict(self.loss_dict,
18                                                            render_results,
19                                                            self.white_bg)
20    
21    @torch.no_grad()
22    def validation(self):
23        self.val_dataset_size = len(self.datapipeline.validation_dataset)
24        for i in range(0, self.val_dataset_size):
25            self.call_hook("before_val_iter")
26            batch = self.datapipeline.next_val(i)

Run pointrix#

We can add our model and dataset implemented above in pointrix framework.

Note

we need to modify the model and dataset name to our names of new model and datasets implemented above. Pointrix framework can find your model and datasets by registry.

We highlight the modified part.#
 1from model import DeformGaussian
 2from dataformat import NerfiesReFormat
 3
 4def main(args, extras) -> None:
 5    
 6    cfg = load_config(args.config, cli_args=extras)
 7
 8    cfg.trainer.model.name = "DeformGaussian"
 9    cfg.trainer.dataset.data_type = "NerfiesReFormat"
10    # you need to modify this path to your dataset.
11    cfg.trainer.dataset.data_path = "/home/clz/data/dnerf/cat"
12    cfg['trainer']['optimizer']['optimizer_1']['params']['deform'] = {}
13    cfg['trainer']['optimizer']['optimizer_1']['params']['deform']['lr'] = 0.00016 * 5.0
14    cfg.trainer.val_interval = 5000
15
16    gaussian_trainer = Trainer(
17        cfg.trainer,
18        cfg.exp_dir,
19    )
20    gaussian_trainer.train_loop()    
21    model_path = os.path.join(
22        cfg.exp_dir, 
23        "chkpnt" + str(gaussian_trainer.global_step) + ".pth"
24    )
25    gaussian_trainer.save_model(model_path)
26    print("\nTraining complete.")
